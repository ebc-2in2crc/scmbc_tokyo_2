branch

branchの作成・削除を行う。
git branch
branch一覧の取得
git branch <branchname>
branchの作成。 <branchname>は作成するbranch名

init

git-init - 空Gitリポジトリを作成 または 既存のものを再初期化します
git init [-q | --quiet] [--bare] [--template=] [--separate-git-dir ] [--shared[=]] [directory]
-q, --quiet 警告とエラーのみを表示します．他のメッセージは破棄します．

--bare
ベアリポジトリを作成します．
ベアリポジトリは作業コピーを持たないリポジトリで，作業を中継するリポジトリとして使用します．

--template=
テンプレートディレクトリを設定します
init時に任意のフックスクリプトをコピーしたい場合に使用します

--separate-git-dir=
任意のディレクトリにリポジトリを設定します
リポジトリとワークツリーを分けたいときに使用します
--shared[=(false|true|umask|group|all|world|everybody|0xxx)] 複数ユーザーでリポジトリを共有する際のアクセス権を設定します．

reset

ワークツリーやインデックスを指定コミットの状態にする
git reset [--soft | --mixed | --hard] []

--soft コミットだけ取り消す
--mixed インデックスの変更を取り消す
--hard ワークツリーとインデックスを破棄して指定コミットの状態にする
